Minna no Nihongo - Хичээл 7

１．Ｎ（багаж хэрэгсэл / хэрэгсэл）で Ｖ（үйл үг）

Тайлбар: Ямар нэгэн зүйлийг багаж, хэрэгсэл эсвэл ямар нэг аргаар хийж байгааг заахад 「で」-г хэрэглэнэ.

  • えんぴつ で かきます。
    → Би харандаагаар бичдэг.
  • バス で いきます。
    → Би автобусаар явдаг.
  • にほんご で てがみ を かきます。
    → Би япон хэлээр захиа бичдэг.
  • メール で てがみ を もらいました。
    → Би мэйлээр захиа авсан.
  • なん で おみず を のみます か。
    → Та усыг юугаар уудаг вэ?

２．「～は ～ご で なん です か」

Тайлбар: Ямар нэгэн үг, өгүүлбэрийг өөр хэл дээр хэрхэн хэлэхийг асуух хэлбэр.

  • [Thank you] は にほんご で なん です か。
    → "Thank you"-г япон хэлээр юу гэдэг вэ?
    ……..ありがとう です。
  • 「さよなら」 は えいご で なん です か。
    → 「さよなら」 англи хэлээр юу вэ?
    ……..Goodbye です。

３．Ｎ（хүн）に Ｖ（あげます、かします、あげます гэх мэт）

Тайлбар: Хэн нэгэнд ямар нэгэн зүйл өгөхөд хэрэглэнэ. 「に」 нь "хэнд" гэдгийг заана.

  • さくらさん に はな を あげます。
    → Би Сакурад цэцэг өгнө.
  • やまださん は ミラさん に ほん を かしました。
    → Ямада сан Миллад ном зээлүүлсэн.
  • せんせい は がくせい に えいご を おしえます。
    → Багш сурагчид англи хэл заана.
  • だれ に おかね を かします か。
    → Та хэнд мөнгө зээлүүлэх вэ?

💡 Анхаар: Байгууллага, газар зааж байгаа бол 「に」 биш, 「へ」 хэрэглэнэ.

  • かいしゃ へ でんわ を かけます。
    → Би компанид утасдана.
  • わたし の だいがく へ にもつ を おくります。
    → Би өөрийн их сургууль руу ачаа илгээнэ.

４．Ｎ（хүн）に Ｖ（もらいます、かります、ならいます）

Тайлбар: Хэн нэгнээс ямар нэгэн зүйл авч байгааг заах хэлбэр. Энд 「に」 нь "хэнээс" гэдгийг заана. Зарим үед 「から」-г ч хэрэглэж болно.

  • サントスさん は さくらさん に プレゼント を もらいました。
    → Сантос Сакурагаас бэлэг авсан.
  • マリアさん は やまださん から おかね を かりました。
    → Мария сан Ямада сангаас мөнгө зээлсэн.
  • わたし は ともだち に にほんご を ならいました。
    → Би найзаасаа япон хэл сурсан.
  • だれ に ほん を かりました か。
    → Та хэнээс ном зээлсэн бэ?

💡 Анхаар: Байгууллагаас авсан бол зөвхөн 「から」 хэрэглэнэ.

  • かいしゃ から この プレゼント を もらいました。
    → Би энэ бэлгийг компаниас авсан.

５．もう Ｖました。

Тайлбар: Аль хэдийн ямар нэгэн үйлдэл хийгдсэн бол 「もう Ｖました」 хэлбэрийг хэрэглэнэ. Харин хийгдээгүй бол 「まだです」 гэж татгалзана.

  • もう さけ を のみました か。
    → Аль хэдийн архи уусан уу?
  • はい、もう のみました。
    → Тийм ээ, уусан.
  • いいえ、まだ です。
    → Үгүй ээ, хараахан болоогүй.
