1. 〜を Vー (үйл үг)ます

Тайлбар:
「を」 нь үйлдлийн объект бүрэү үйл үгний үйлдэл ямар зүйлд хийгдэж байгааг заана.

  • ぱん を たべます → Би талх иднэ.
  • おちゃ を のみます → Би цай ууна.
  • てがみ を かきます → Би захиа бичнэ.

2. なに を Vー ます か

Тайлбар:
Юу хийх вэ гэж асуухдаа 「なに を」 гэж хэрэглэнэ.

  • なに を のみます か → Та юу уух вэ?
  • じ\uхс を のみます → Би жүүс ууна.

3. どこ で Vー ます か

Тайлбар:
Үйлдэл хаана болж байгааг 「どこ で」 гэж асууна. 「で」 нь үйлдэл болж буй газар заана.

  • どこ で ひるごはん を たべます か → Та хаана өдрийн хоол иддэг вэ?
  • れすとらん で たべます → Би ресторанд иддэг.

4. いっしょ に Vー ません か

Тайлбар:
Хамтдаа ямар нэг зүйл хийхийг санал болгох найрсаг хэлбэр. 「〜ましょう」-с арай зөөлөн уриалга.

  • いっしょ に こーひー を のみません か → Хамтдаа кофе уух уу?
  • いいですね → Сайхан байна шүү!

5. 〜 ちょっと...

Тайлбар:
Япон хэлэнд шууд “үгүй” гэж хэлэх нь бүдүүлэг тул “〜ちょっと...” гэж зөөлөн татгалзахад хэрэглэнэ.

  • いっしょ に えいが を みません か → Хамтдаа кино үзэх үү?
  • すみません。きょう は ちょっと... → Уучлаарай, өнөөдөр жаахан...

6. いつ Vー ます か

Тайлбар:
Үйлдэл хэзээ болохыг асуухдаа 「いつ」 хэрэглэнэ.

  • いつ とうきょう へ いきます か → Хэзээ Токио руу явах вэ?
  • あした いきます → Маргааш явна.

7. なん で Vー ます か

Тайлбар:
Үйлдэл юугаар/юуны тусламжтай хийгдэж байгааг асуухад 「なん で」 хэрэглэнэ.

  • なん で とうきょう へ いきます か → Токио руу яаж очих вэ?
  • しんかんせん で いきます → Шинкансэнээр явна.

8. だれ と Vー ます か

Тайлбар:
Хэнтэй хамт үйлдлийг хийж байгааг асуухдаа 「だれ と」 хэрэглэнэ.

  • だれ と とうきょう へ いきます か → Хэнтэй хамт Токио руу явах вэ?
  • ともだち と いきます → Найзтайгаа явна.
