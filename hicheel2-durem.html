1. これ・それ・あれ (Заах төлөөний үгс)

➡ これ は ～ です。(Энэ бол ~ юм.)

➡ それ は ～ です。(Тэр бол ~ юм.)

➡ あれ は ～ です。(Тэр (хол байгаа) бол ~ юм.)

✔ これは本です。 (Энэ бол ном.)

✔ それは辞書ですか？ (Тэр толь бичиг үү?)

✔ あれは私のかばんです。 (Тэр (хол байгаа) миний цүнх.)

2. この・その・あの + нэр үг

➡ この + нэр үг (энэ ~)

➡ その + нэр үг (тэр ~)

➡ あの + нэр үг (тэр (хол байгаа) ~)

✔ この本はおもしろいです。 (Энэ ном сонирхолтой.)

✔ その時計はいくらですか？ (Тэр цаг хэд вэ?)

✔ あの人は誰ですか？ (Тэр хүн хэн бэ?)

3. そうです / そうじゃありません (Тийм / Тийм биш)

➡ ～ は ～ ですか？ (Асуух өгүүлбэр)

➡ はい、そうです。 (Тийм.)

➡ いいえ、そうじゃありません。 (Үгүй, тийм биш.)

✔ これはボールペンですか？ (Энэ бал уу?)

➡ はい、そうです。 (Тийм.)

✔ あれはあなたのノートですか？ (Тэр (хол байгаа) таны дэвтэр үү?)

➡ いいえ、そうじゃありません。 (Үгүй, тийм биш.)

4. なん (何 - Юу?)

➡ これ/それ/あれ は なん ですか？ (Энэ/тэр юу вэ?)

✔ これは何ですか？ (Энэ юу вэ?)

➡ それは辞書です。 (Тэр бол толь бичиг.)

✔ あの箱の中には何がありますか？ (Тэр хайрцагны дотор юу байна?)

➡ 本があります。 (Ном байна.)
